========================
|emod_api| documentation
========================

|emod_api| is the interface for |EMOD_l| that users of |IT_s| interact with to create and modify |EMOD_s| simulations.  Additional functionality 
for interacting with |EMOD_s| is provided in the 
:doc:`emodpy:emodpy` and :doc:`idmtools:idmtools_index`.

See :doc:`idmtools:index` for a diagram showing how |IT_s| and each of the
related packages are used in an end-to-end workflow using |EMOD_s| as the
disease transmission model.
